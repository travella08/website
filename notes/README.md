# Notes

Drop LaTeX-compiled PDFs here (e.g. `math417-group-theory.pdf`), then add
an entry to the `notes` array in `js/data.js`:

```js
{
  title: "Group Theory — Lecture Notes",
  course: "MATH 417",       // or any label, e.g. "Misc"
  date: "2026-09-12",       // used for newest-first sorting
  description: "Optional one-line summary.",
  url: "notes/math417-group-theory.pdf",
},
```

The Notes section on the homepage renders them automatically.
